   - Bukan error, bukan bug

2. **Dokumentasi mungkin menggunakan versi berbeda**
   - Dokumentasi online mungkin menunjukkan versi Keenicons yang lebih baru/lama
   - Atau menggunakan icon library lain (Font Awesome, Material Icons, dll)

3. **Keenicons punya style minimalis**
   - Outline icons: Garis tipis, minimalis
   - Solid icons: Filled, lebih tebal
   - Duotone icons: Dua warna

## 📝 CARA MENGGUNAKAN ICON

### **1. Outline Icons** (Paling Umum)
```html
<i class="ki-outline ki-picture"></i>
<i class="ki-outline ki-check"></i>
<i class="ki-outline ki-brifecase-tick
"></i>
```

### **2. Solid Icons** (Untuk Emphasis)
```html
<i class="ki-solid ki-check"></i>
<i class="ki-solid ki-home"></i>
```

### **3. Duotone Icons** (Two-Tone)
```html
<i class="ki-duotone ki-home">
    <span class="path1"></span>
    <span class="path2"></span>
</i>
```

### **4. Dengan Size**
```html
<i class="ki-outline ki-picture fs-1"></i>  <!-- Largest -->
<i class="ki-outline ki-picture fs-2"></i>
<i class="ki-outline ki-picture fs-3"></i>
<i class="ki-outline ki-picture fs-4"></i>  <!-- Smallest -->
```

### **5. Dengan Color**
```html
<i class="ki-outline ki-check text-success"></i>
<i class="ki-outline ki-arrow-zigzag text-danger"></i>
<i class="ki-outline ki-information text-primary"></i>
```
