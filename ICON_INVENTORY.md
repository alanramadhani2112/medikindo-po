## 🎨 Format & Ukuran Icon

### Format Standar
```html
<i class="ki-solid ki-{icon-name} fs-{size}"></i>
```

### Ukuran yang Digunakan

| Size Class | Ukuran | Penggunaan |
|------------|--------|------------|
| `fs-1` | Extra Large | Hero icons |
| `fs-2` | Large | Button icons, menu icons |
| `fs-3` | Medium | Inline icons, dropdown |
| `fs-4` | Small | Table actions, badges |
| `fs-2x` | 2x | Dashboard cards |
| `fs-3x` | 3x | Empty states |

### Warna Icon

| Color Class | Warna | Penggunaan |

## 📝 Catatan

1. **Semua icon menggunakan format Keenicons** dari Metronic 8
2. **Format konsisten**: `ki-solid ki-{name}`
3. **Total 72 icon unik** digunakan di seluruh sistem
4. **Icon paling banyak**: Action buttons (check, cross, plus, pencil, trash)
5. **Icon paling sedikit**: Time & Date (hanya 3 icon)

## 🎯 Rekomendasi

Jika ingin mengganti icon, pertimbangkan:

1. **Konsistensi**: Gunakan icon dari library yang sama
2. **Ukuran**: Pertahankan ukuran yang sudah ada (fs-2, fs-3, fs-4)
3. **Warna**: Pertahankan color scheme yang sudah ada
4. **Semantik**: Icon harus sesuai dengan fungsinya
5. **Accessibility**: Pastikan icon mudah dipahami
