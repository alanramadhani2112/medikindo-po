- **Structure**: 6-part layout (header, KPI, filter, tabs, table, pagination)

- **Buttons**: primary, success, danger, light

## 🎯 SUCCESS CRITERIA

A module is considered standardized when:

✅ Uses `@extends('layouts.app')`
✅ Follows 6-part structure
✅ Uses standard table classes
✅ Has pagination with count
✅ Has filter with reset button
✅ Uses Keenicons only
✅ Follows spacing rules
✅ Uses correct button variants
✅ Uses correct badge colors
✅ Has professional empty state
✅ No custom components
✅ No diagnostics errors
